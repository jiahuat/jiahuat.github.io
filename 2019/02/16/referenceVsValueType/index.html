Reference vs Value Types in Swift

Posted on 2019-02-16   |   In iOS

导语

Swfit中的structures 和 enumerations可以完成大部分class的功能，在我们大量使用它们的时候，我们有必要清楚究竟发生了些什么，这样才能更好的掌控自己的程序。

Swift中的Value Types 和 Reference

Value type 是在赋值给变量或者常量，或者在传递给函数时使用复制的方式；
Reference则不是复制，而是引用相同的实例。
在Swift中，integers, floating-point numbers, Booleans, strings, arrays ，dictionaries， structures and enumerations是Value type；class是Reference Type.

Copy-on-Write

Copy-on-Write是Value types的一个强大的特性，值得是在最开始赋值的时候，并没有发生拷贝，而是共用同一块地址。只有在修改其中某个的值的时候才发生拷贝。下面通过一个例子来说明：

struct Person {
  var name: String
  var gender: String
  var phoneNumber: String
  var email: String
}

var person1 = Person(name: "ZhaoSi", gender: "male", phoneNumber: "123", email: "heiheihei.com")
var person2 = person1
var person3 = person2
