Swift中的Value Types 和 Reference

Value type 是在赋值给变量或者常量，或者在传递给函数时使用复制的方式；
Reference则不是复制，而是引用相同的实例。
在Swift中，integers, floating-point numbers, Booleans, strings, arrays ，dictionaries， structures and enumerations是Value type；class是Reference Type.

Copy-on-Write

Copy-on-Write是Value types的一个强大的特性，值得是在最开始赋值的时候，并没有发生拷贝，而是共用同一块地址。只有在修改其中某个的值的时候才发生拷贝。下面通过一个例子来说明：
